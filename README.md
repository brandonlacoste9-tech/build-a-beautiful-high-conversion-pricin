## Quick start (local / WSL)

```bash
git clone <your-repo-url>
cd build-a-beautiful-high-conversion-pricin
npm install
cp .env.example .env.local
# set DATABASE_URL (and OpenAI keys if using the agent route)
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
